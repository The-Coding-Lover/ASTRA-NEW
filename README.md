# Astra Chatbot

**Astra** is a smart, voice-enabled AI chatbot designed for web browsers. It features natural language understanding, voice recognition, and text-to-speech synthesis. Astra can perform various tasks such as answering questions, setting alarms, managing a to-do list, running a planner, and much more â€” all through an interactive and visually engaging interface.

---

## Features

- **Voice Input & Output**
  - Speech Recognition and Text-to-Speech capabilities
  - Real-time audio conversation with the chatbot

- **Core Functionalities**
  - Time and Date queries
  - Calculator operations
  - Countdown timer and alarms
  - To-do list manager and daily planner
  - Fun quizzes and fact sharing
  - Theme switching (light/dark)
  - Voice customization (male/female voices)

- **Advanced Capabilities**
  - Learns from conversations
  - Remembers user preferences using local storage
  - Responsive full-screen UI with animated AI character
  - Personality-based responses

---

## Technologies Used

- **Frontend:**
  - HTML5, CSS3, JavaScript (Vanilla JS)
  - Web Speech API (for voice input/output)
  - LocalStorage API
  - Responsive design principles

- **Optional Enhancements:**
  - Web Animations API / Lottie for character animation
  - Custom themes using CSS variables
  - Modular JS structure
